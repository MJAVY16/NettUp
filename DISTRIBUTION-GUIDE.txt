IMPORTANT NOTES FOR TESTERS:
-----------------------------
1. WINDOWS SMARTSCREEN WARNING
   - Since the app is unsigned, Windows will show a security warning
   - This is NORMAL for unsigned apps
   - Testers should click "More info" then "Run anyway"
   - Alternative: Right-click installer → Properties → Check "Unblock" → Apply

2. LICENSE ACTIVATION
   - Each tester needs ONE license key from production-keys.txt
   - Format: XXXX-XXXX-XXXX-XXXX
   - Each key works on ONE machine only
   - Keys are in: production-keys.txt (lines 7-106)

3. INSTALLATION STEPS
   - Run the installer
   - Choose installation directory
   - Wait for installation to complete
   - Launch Financial Tracker
   - Enter license key when prompted
   - Start using the app!
